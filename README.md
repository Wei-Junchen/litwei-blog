# Litwei Blog

轻量 Hugo 静态博客：Markdown 写作，GitHub Actions 构建 HTML，VPS 只用 Nginx 托管。

## 本地预览

```bash
./scripts/preview.sh
```

## 新建文章

```bash
./scripts/new-post.sh tech "My Tech Note"
```

## 构建

```bash
./scripts/build.sh
```

## Formspree

把 `hugo.toml` 里的 `https://formspree.io/f/YOUR_FORM_ID` 替换成你的 Formspree endpoint。
