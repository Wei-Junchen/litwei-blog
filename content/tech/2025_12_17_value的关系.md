# lvalue,xvalue,prvalue的关系

## lvalue

- 表示可以寻址的左值对象
- `T`和`T&`类型, 左侧的` T&&` 类型, 以下的`x`均为lvalue
  - `T x = 100`
  - `T& x = y`
  - `T&& x = 1000 //右值引用类型的命名变量也是lvalue`  

## xvalue

- 也称为将亡值(expiring value)
- 一般通过`std::move`获取的值即为这种类型，或者函数体返回的`T&&`也为xvalue

## prvalue

- 字面量 (`123 : int `,` "Hello" : string`)
- = 右侧初始化对象 (`T x = T{}`,`T x= T{"Hello",123}`, `T&& x = T{123}`)
  - 在standard < 17时, `T{}` 这类 = 右侧初始化实质上是通过构造一个匿名对象然后通过`std::move`移动初始化的
  - 在standard >= 17时, `T{}` 右侧初始化直接在左侧对象上进行初始化，不会进行移动构造
  - 对于 `T&& x = T{...}`, 右侧的`T{...}`也为prvalue，因为其并不能实际寻址, 实际上也是到`x`上进行构造初始化
