# shell快捷键学习
- ctrl + A : 移动光标至段首
- ctrl + E : 移动光标至段尾 E:end
- ctrl + B : 光标往段首移动一格 B:back
- ctrl + F : 光标往段尾移动一格 F:front
- ctrl + U : 删除段首到当前光标的所有内容
- ctrl + K : 删除当前光标到段尾的所有内容
- ctrl + P : 上一条指令 P:previous
- ctrl + N : 下一条指令 N:next
- ctrl + L : 清屏
- ctrl + H : 删除当前光标的字符
- ctrl + W : 删除当前光标前的一个词 W:word
- ctrl + Z : 挂起某一个进程 (bg:挂到后台,fg:恢复)
- ctrl + R ：搜索历史指令
- ctrl + Q : 退出搜索历史指令
- ctrl + D : 结束当前shell会话
- ctrl + C : 结束当前进程
- ctrl + S : 冻结shell输出
- ctrl + Q : 恢复shell输出
- ctrl + Y : 粘贴刚才删除的内容
